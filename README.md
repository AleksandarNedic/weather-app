# Weather App 🌤️

A modern weather application built with JavaScript, Webpack, Axios, and WeatherAPI.

The application allows users to search for weather information in any city, automatically detects the user's current location, displays a 7-day weather forecast, and provides a weather forecast for the date 30 days from today.

## Features

- 🌍 Automatic user geolocation
- 🔍 Search weather by city name
- 🌡️ Current temperature and weather conditions
- 📅 7-day weather forecast
- 🗓️ Weather forecast for the date 30 days from today
- 🌙 Automatic day/night theme based on local time
- 🎨 Dynamic UI theme switching
- 📍 Display location information
- ⚡ Fast API requests using Axios
- 🔒 API key stored in `.env`
- 📦 Bundled with Webpack

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Axios
- WeatherAPI
- Webpack
- npm
